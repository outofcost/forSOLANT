# Game Management — Full QA Testing
Environment: DEV  
Product: SG Admin Panel
---

## Description

Провести полное тестирование модулей Game Management в админ-панели, чтобы подтвердить корректность управления играми, провайдерами, коллекциями, темами и агрегаторами, а также корректную синхронизацию с фронтом без задержек и ошибок отображения.

---

## Scope / Area of Testing

SG Admin Panel (DEV)  

(актуальная Figma)

## Main Modules

### Collections

- Создание / редактирование / удаление коллекций
- Типы коллекций:
  - Standard / Main
  - Collection Bar
  - Game Page
- Независимый Order для каждого типа
- Способы наполнения:
  - поштучно (отдельные игры)
  - по провайдеру
  - по категории

- Счётчики количества игр
- Реакция на выключение игры / категории
- Иконки / Inside Bar:
  - загрузка
  - приоритет кастомных
  - кеш-инвалидация
- Фильтры:
  - по провайдеру
  - по статусу
  - комбинированные
  - сброс фильтров

## Acceptance Criteria

- Все изменения в админке отображаются на фронте без ручного сброса кеша
- Счётчики и фильтры консистентны на всех уровнях
- Все ошибки отображаются человекочитаемо:
  - валидации
  - загрузка изображений
- Нет критических / блокирующих багов
- Все дефекты заведены и привязаны к данному эпику
- Подготовлен краткий отчёт с ссылками на видео и скриншоты

## Test Blocks

### Block 1: Provider Management

- Провайдер отключается синхронно везде
- Игры провайдера:
  - остаются в коллекциях
  - статусы не меняются
- Отключённые игры:
  - исчезают из поиска на фронте
  - скрыты, но остаются в коллекциях

---

### Block 2: Image Management

- Загрузка кастомных изображений
- Кастомное изображение имеет приоритет над изображением провайдера
- Успешно загружается только формат 200×200
- Изменения отражаются:
  - в таблице коллекций
  - в games list одновременно

---

### Block 3: Soft Switch Provider

- Soft Switch отключается при выключении агрегатора
- Feature Groups Basic / Pro синхронизированы
- Недоступные группы не отображаются в games list

---

### Block 4: Game Counters

- Счётчик отражает корректные значения (например: 178 enabled / 0 disabled)
- При выключении игры счётчик уменьшается на 1
- Недоступные группы полностью отфильтрованы

---

### Block 5: Collections Management

#### Collection Types

- Standard Collection:
  - главная страница
  - 1–2 линии
  - собственный Order
- Collection Bar:
  - категории (Mega Way, Classic)
  - только Order и иконки
- Game Page:
  - собственный Order
  - настройка линий

#### Game Adding Methods

- Поштучно (одна конкретная игра)
- Через провайдера (все игры провайдера)
- Через категорию (источник сохраняется в UI)

#### Critical Checks

- Order отличается для каждого типа коллекции
- Выключение игры работает глобально
- Таблица коллекций и games list показывают одинаковые значения
- Фильтры работают во всех местах

---

### Block 6: Themes Management

- Порядок тем соответствует Order
- Внутри каждой темы:
  - собственный Order
  - не пересекается с другими темами
- Подтемы:
  - отображаются
  - сохраняют связь с родительской темой
- Добавление через категорию сохраняет источник
